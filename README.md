# DespensaDigital

## Descripción
DespensaDigital es una aplicación móvil que permite gestionar 
la mercadería presente en los hogares, ayudando a reducir el 
desperdicio de alimentos y su impacto ambiental. La propuesta 
busca además mejorar la economía familiar mediante un control 
eficiente del inventario doméstico.
